# Security policy

## Supported versions

Security fixes are provided for the latest published Hop release.

## Reporting a vulnerability

Please do not open a public issue for a suspected vulnerability. Email
<hello@gnosyslabs.xyz> with `[Hop security]` in the subject and include the
affected version, impact, reproduction steps, and any suggested mitigation.

We will acknowledge the report privately, coordinate remediation and
disclosure with the reporter, and credit the reporter unless they prefer to
remain anonymous.
